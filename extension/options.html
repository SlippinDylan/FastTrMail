FastTrMail

给网页邮箱正文加一个内嵌翻译按钮。点击邮箱工具区里的“翻译”后，译文会追加在原文下方，不覆盖原文。

总设置

这里决定扩展默认使用哪个翻译服务，以及翻译成哪种目标语言。

Google Cloud API 配置

如果总设置里选择了 Google Cloud API，这里的 API Key 就会被用于翻译请求。

官方开通说明 官方 API Key 管理 打开 Google API 控制台

Microsoft API 配置

如果总设置里选择了 Microsoft Translator API，这里的 Key 和 Region 会用于翻译请求。

官方申请与快速开始 打开 Azure Portal
